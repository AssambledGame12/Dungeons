# 🏰 Dungeons

Un roguelite dungeon crawler 2D desarrollado completamente en HTML, CSS y JavaScript vanilla. Sin dependencias externas — corre en cualquier navegador.

**[🎮 Jugar ahora](https://assambledgame12.github.io/Dungeons/)**

---

## ¿Qué es?

Juego de exploración y combate top-down donde sobrevives oleadas de enemigos en un calabozo, recolectas experiencia, subes de nivel y eliges mejoras que cambian tu estilo de juego.

## Características

- 5 tipos de enemigos con comportamientos y formas distintas (Esqueleto, Demonio, Golem, Araña, Liche)
- Sistema de oleadas con dificultad escalable por sala
- Sistema de niveles con elección de 3 perks aleatorios al subir
- 10+ mejoras disponibles: Multidisparo, Penetración, Vampirismo, Explosivo, y más
- Tienda in-game con oro dropeado por enemigos
- Minimapa en tiempo real
- Números de daño flotantes
- Música de ambiente generada proceduralmente con Web Audio API
- Sin instalación, sin internet después de cargar

## Controles

| Acción | Tecla |
|--------|-------|
| Mover | `W A S D` |
| Disparar | `Espacio` |

## Tecnologías

- HTML5 Canvas API (renderizado del juego)
- Web Audio API (música procedural)
- JavaScript vanilla (lógica del juego, IA de enemigos, física)
- CSS3 (UI y overlays)
